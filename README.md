A simple and lightweight Angular (12+) Carousel Component.

## Installation

Install `@marcreichel/angular-carousel` with npm:

```bash
npm install @marcreichel/angular-carousel --save
```

or with yarn:

```bash
yarn add @marcreichel/angular-carousel
```
